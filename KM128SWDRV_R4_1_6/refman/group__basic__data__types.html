Basic Data Types

Overview

The MKMxxZxxACxx5 bare metal software drivers defines several basic data types to support code portability between different hardware architectures and tools. These basic data types, which are defined in the C header file common.h supports generic word and integer data types. This is used throughout the interface definitions for the bare metal software drivers.
