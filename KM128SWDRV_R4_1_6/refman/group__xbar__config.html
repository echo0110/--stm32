XBAR Configuration Structures
XBAR Driver

Overview

This section describes default configuration structures for XBAR module. Create a new configuration structure if default ones don't initialize on-chip peripheral in required operating mode (see Creating Configuration Structure).

Macros

#define XBAR_MODULE_NO_EDGE_DETECT_CONFIG
 Edge detection on the XBAR_DMA output channel disabled. Interrupt and DMA disabled on the peripheral level. More...
 
#define XBAR_MODULE_POSEDGE_DETECT_CONFIG
 Enabled rising edge detection on the XBAR_DMA output channel. Interrupt enabled but DMA disabled on the peripheral level. More...
 
#define XBAR_MODULE_NEGEDGE_DETECT_CONFIG
 Enabled falling edge detection on the XBAR_DMA output channel. Interrupt enabled but DMA disabled on the peripheral level. More...
 
#define XBAR_MODULE_ANYEDGE_DETECT_CONFIG
 Enabled rising and falling edge detection on the XBAR_DMA output. Interrupt enabled but DMA disabled on the peripheral level. More...
