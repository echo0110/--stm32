Release Notes

REVISION 4.1.6 (Nov.15,2016):

  • Modified FreeRTOS kernel to disable all interrupts prior entry to critical section and enable all interrupts upon exiting from critical section. This kernel behavior is fully compatible with standard FreeRTOS port to the ARM Cortex-M0 core. All freertos_cfg header files updated to reflect kernel change.
  • Updated PLL_Disable macro and Quad Timer driver.
  • Added UART_SetBaudRate macro.
  • Removed RCM_ClrResetFlags macro.
  • Fixed issue of generating callback events after conversion for these ADC channels with interrupt disabled.

REVISION 4.1.5 (Apr.22,2016):

  • Added configuration structures for watchdog timer operating in debug mode.
  • Included IAR EWARM 7.60.1 project templates.
  • Modified LLWU Pin Falling Edge Wakeup example.

REVISION 4.1.4 (Mar.31,2016):

  • Added SVC_InstallCallback to install Supervisor Call (SVC) callback function (replaces SVC_GetId macro).
  • Added SVC Example.
  • Fixed type conversion issue in ADC driver.

REVISION 4.1.3 (Feb.21,2016):

  • Fixed FreeMASTER recorder cannot compile using Atollic TrueSTUDIO.
  • Fixed DMA_Init function cannot re-initiate next data transfer.
  • Added DMA_CH_HWTRG_M2M_CN_CONFIG and DMA_CH_HWTRG_M2M_CS_CONFIG configuration structures.

REVISION 4.1.2 (Jan.27,2016):

  • Fixed setting in KDS projects and project template.

REVISION 4.1.1 (Nov.23,2015):

  • Fixed IAR EWARM FreeRTOS project templates and projects.
  • Fixed minor issues in AFE and UART drivers.

REVISION 4.1.0 (Nov.17,2015):

  • Included Kinetis Design Studio 3.0.0 support.
  • Included Rowley CrossWorks 3.6.0 support.
  • Changed VREF Driver configuration structures.

REVISION 4.0.0 (Jun.22,2015):

  • Unified driver's API for MKM34Z256VLx7 and MKMxxZxxACxx5 microcontroller series.
